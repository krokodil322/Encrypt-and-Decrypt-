Об программе:
	Представляет собой шифровщик\расшифровщик текстовых файлов у
	которого есть полноценное GUI оформление(на Tkinter) с окном отображения
	текста, кнопками открытия\создания\сохранения текстовых файлов.

	При первом запуске попросит ввести новый пароль.

	Файлы:
		launcher.exe - Все файлы ниже в этом экзешнике, он и запускает программу.

		cipher2.py - Все ф-ии шифровщика.
		tkinter_window_form.py - Классы GUI окна.
		craft.py - Ф-я генерации ключа шифрования и ф-я хэширования пароля.
		launcher.py - Запуск программы(через консоль)
